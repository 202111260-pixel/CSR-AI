🌐 Frontend Layer

React 19 SPA built with Vite 6.3. All 24 pages are React.lazy() loaded. Tailwind CSS v4 for styling. Zustand stores persist auth state and UI preferences.

🛡️ Security Chain

9-layer middleware pipeline processes every request. Helmet sets 11 security headers. CSRF uses double-submit cookie pattern. JWT lives in httpOnly cookies — never in JS.

⚡ API Layer

17 Express routers serve 94+ endpoints. Zod v3 validates all inputs. ActivityLog tracks every write operation. Standard response format across all endpoints.

🐘 Data Layer

PostgreSQL 17 via Prisma ORM. 20 models with cascading deletes, indexed FKs, UUID PKs. RefreshToken stores SHA-256 hashes for secure token rotation.

🤖 AI Integration

GitHub Models API powers bilingual analysis with 14 LLM models. Multi-model retry pool with automatic fallback to local analysis engine.

🔄 Data Flow

Browser sends httpOnly cookies automatically. Axios injects CSRF token header. 401 responses trigger silent refresh with request queuing. No tokens visible to JavaScript.
